# Photo Folders Guide

## 📁 Folder Structure

```
images/
├── backgrounds/    ← Add your BACKGROUND photos here
└── main/          ← Add your MAIN gallery photos here
```

---

## 🎨 BACKGROUND PHOTOS
**Location:** `images/backgrounds/`

Add background images for each section:
- `hero.jpg` - Hero/Welcome section (recommended: 1200x800 or larger)
- `message.jpg` - Message section
- `gallery.jpg` - Photo gallery section background
- `wishes.jpg` - Birthday wishes section
- `surprise.jpg` - Surprise section
- `closure.jpg` - Closure/Final section

**Recommended:** Landscape images (wide format) that can be blurred nicely

---

## 📸 MAIN GALLERY PHOTOS
**Location:** `images/main/`

Add your favorite couple photos:
- `photo1.jpg` - First gallery photo
- `photo2.jpg` - Second gallery photo
- `photo3.jpg` - Third gallery photo

**Recommended:** Square or near-square images (or any size - they'll auto-fit)

---

## 📝 Instructions

1. **Add your photos** to the appropriate folders
2. **Match the filenames** exactly as mentioned above (or update the code if you use different names)
3. **The code is already configured** to look for images in these folders
4. **Reload the website** to see your changes

---

## 🌟 Tips

- Use high-quality images for best visual effect
- Background photos are best with soft, scenic, or partner photos
- Main gallery photos can include selfies, couple shots, or special moments
- Keep file sizes reasonable (under 2MB each) for fast loading
