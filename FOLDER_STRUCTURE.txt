=====================================

   ✗ "random-name.jpg" won't work

Main Gallery Images:
   ✓ photo1.jpg, photo2.jpg, photo3.jpg
   ✗ "image1.jpg" = needs exact names

Image Format:
   ✓ JPG, PNG, WebP
   ✓ Any resolution works
   ✓ Landscape best for backgrounds

File Size:
   ✓ Keep under 2MB each for fast loading
   ✓ Compress if needed (tools: tinypng.com, compressor.io)
